# Password Strength Checker (Python)

A standalone Python script that evaluates password strength using entropy calculations, character-set analysis, dictionary detection, and pattern-based weakness checks.  
Scores range from **0 to 100**, with strength levels from **Very Weak** to **Very Strong**.

---

## Features
- Entropy calculation  
- Common-password detection  
- Dictionary word detection  
- Repeated-character checks  
- Pattern recognition (only numbers, only letters, etc.)  
- Character-variety scoring  
- Length-based scoring  
- Single-file script  
- Fully local and offline  

---

### Run the script
